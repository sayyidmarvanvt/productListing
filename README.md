
## Installation

1. Clone the repository:
   ```
   git clone https://github.com/sayyidmarvanvt/productListing.git
   ```

2. Install the dependencies:
   ```
   npm install
   ```

3. Start the application:
   ```
   npm start
   ```

## Usage

1. Open your browser and navigate to `http://localhost:3000`.

2. You should see the product list page.

3. Use the search bar to filter the products by name.

4. Sort the products by price using the dropdown menu.

5. Click on the "View Details" button to view the details of a specific product.

## Features

- Display a list of products
- Search for products by name
- Sort products by price
- View details of a specific product
